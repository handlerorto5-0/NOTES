ELK расшифровывается как elasticsearch, logstash и kibana для широкого спектра задач сбора, хранения и анализа данных.

logstash – это утилита для сборки, фильтрации и последующего перенаправления в конечное хранилище данных.
input {} filter {} output {}

elasticsearch – это решение для полнотекстового поиска, построенное поверх Apache Lucene, но с дополнительными удобствами, типа лёгкого масштабирования, репликации и пр.

Angular.js приложение kibana, позволяющее брать\искать данные по elasticsearch и строить множество красивых графиков
