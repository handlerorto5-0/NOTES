read записывает ввод
$REPLY выводит ввод из read
---------------------
read -p "May I ask your name: " name
echo "Hello $name"
--------------------
ограничение количества введёных символов
read -n1 -p "Press any key to exit"
echo -чтобы строки не скакали
--------------------
Управление видимостью введенного текста
read -sn1 -p "Press any key to exit"
---------------------
передача параметров $1 $2 $3 ...
/code/scripts/option.sh
общепринятые стандарты для опций:
-а: перечислить все элементы
-с: подсчитать все предметы
-d: выходной каталог
-e: развернуть элементы
-f: указать файл
-h: Показать страницу справки
-i: игнорировать регистр символов
-l: перечислить текст
-o: отправить вывод в файл
-q: молчать; не спрашивайте пользователя
-r: обрабатывать что-то рекурсивно
-s: использовать скрытый режим
-v: использовать подробный режим
-x: указать исполняемый файл
-y: принять без подсказки
