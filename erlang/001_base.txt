Атомы:

atom.
atoms_rule@erland.
'Atoms can be cheated'.
atom = 'atom'.

Атом должен бытьв ('') если он не начинается со строчной буквы, или содепжит либые другие символы, кроме букв, цифр, (_), @.

Булева алгебра:

true and false.     - false
false or true.      - true
true xor false.     - true
not false.          - true
not(true and true). - false

andalso, orelse быстрые операторы, провеляют правый аргумент если он нужен

Проверка равенства, неравенства  =:=, =/=
5 =:= 5.   - true точное сравнение
1 =/= 0.   - true
5 == 5.0.  - true  не точное сравнение
5 /= 5.0.  - false

Операторы <,>, >=, =< (=< меньше-равно перевёрнут !!!)

Кортежи - способ организовать данные и сгруппировать множество значений, когда точно известно их количество.
{Элемент1, Элемент2, ... , ЭлементN}
кортеж с тегом, содержит атом и один дополнительный элемент
{point,{X,Y}}.

Списки:
[Элемент1, Элемент2, ... , ЭлементN]
[1,2,3, {numbers, [4,5,6]}, 5.34, atom]
[97,98,99]. вернёт "abc"
Erlang будет печатать списки чисел как сисла, когда хотябы одно из низ не может представлять букву !!!

++, -- склеивание, удаление из списка операторы 
[1,2,3]++[4,5]
[1,2,3] -- [1,2] -- [2] вергёт [2,3] выполняются справа налево

голова списка - первый элеьент
hd([1,2,3,4]). -> [1]
хвост списка - остаток после головы
tl([1,2,3,4]). -> [2.3.4]
