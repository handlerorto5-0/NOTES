ng help
Доступные команды:
  add Добавляет поддержку внешней библиотеки в ваш проект.
  analytics Настраивает сбор метрик использования Angular CLI. См. Https://v8.angular.io/cli/usage-analytics-gathering.
  build (b) Компилирует приложение Angular в выходной каталог с именем dist / по заданному выходному пути. Должен выполняться из каталога рабочей области.
  deploy (d) Вызывает построитель развертывания для указанного проекта или для проекта по умолчанию в рабочей области.
  config Получает или задает значения угловой конфигурации в файле angular.json для рабочей области.
  doc (d) Открывает официальную документацию Angular (angular.io) в браузере и выполняет поиск по заданному ключевому слову.
  e2e (e) Создает и обслуживает приложение Angular, затем запускает сквозные тесты с использованием Protractor.
  generate (g) Генерирует и / или изменяет файлы на основе схемы.
  help Перечисляет доступные команды и их краткие описания.
  lint (l) Запускает инструменты рисования для кода приложения Angular в данной папке проекта.
  new (n) Создает новое рабочее пространство и начальное приложение Angular.
  run Запускает цель Architect с необязательной настраиваемой конфигурацией компоновщика, определенной в вашем проекте.
  serve (s) Создает и обслуживает ваше приложение, перестраивая изменения файла.
  test (t) Запускает юнит-тесты в проекте.
  update Обновляет ваше приложение и его зависимости. Смотрите https://update.angular.io/
  version (v) Выводит угловую версию CLI.
  xi18n Извлекает сообщения i18n из исходного кода.
